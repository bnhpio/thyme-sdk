# Getting Started

## Prerequisites

- Node.js 18+
- Bun (latest)
- Git

## Install Tooling

```bash
bun add -g @bnhpio/thyme-cli
thyme --help
```

## Create a Project

```bash
thyme create hello-thyme
cd hello-thyme
bun install
cp .env.example .env
```

Fill `.env` with RPC URLs, deploy endpoint, and placeholder secrets. Keep the file out of git.

## Generate a Function

```bash
thyme create ping-automation
```

Update `functions/ping-automation/schema.ts` and `args.json` to match your use case. Implement logic inside `index.ts` using the [Thyme SDK](/thyme-sdk).

## Simulate

```bash
thyme simulate ping-automation --profile sepolia-test
```

✅ You now have a full loop: schemas → runner → sandbox → onchain simulation. Continue to the [CLI docs](/cli) for detailed command references or the [CLI + SDK workflow](/cli-sdk) for an end-to-end playbook.
