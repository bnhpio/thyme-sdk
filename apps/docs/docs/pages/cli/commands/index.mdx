# Commands

The CLI exposes four primary commands. Each subpage includes Bun-friendly usage snippets, option tables, execution flow, and best practices.

| Command | Purpose | Typical Stage |
| --- | --- | --- |
| [`thyme create`](/cli/commands/thyme-create) | Scaffold projects or functions | Day 0 / feature kick-off |
| [`thyme auth`](/cli/commands/thyme-auth) | Store auth tokens in `.env` | Once per environment |
| [`thyme simulate`](/cli/commands/thyme-simulate) | Sandbox & onchain validation | CI + pre-merge |
| [`thyme upload`](/cli/commands/thyme-upload) | Package and deploy bundles | Release pipeline |

## Global Flags

- `--env <path>`: load environment variables from a specific file (defaults to `.env`)
- `--debug` *(planned)*: emit verbose logs for sandbox/onchain stages
